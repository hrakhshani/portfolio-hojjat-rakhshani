# AI Researcher Portfolio

This project is a minimal, responsive personal website and portfolio.

- Built with React and Next.js
- Modern, clean design
- Fully responsive
- Easy to customize content in code

## Getting Started

1. Clone the repository  
2. Install dependencies:  
   ```
   npm install
   ```
3. Run the development server:  
   ```
   npm run dev
   ```

Customize your content in the `components/` folder.
